O que é JavaScript?

JavaScript é a linguagem de programação usada para tornar páginas web interativas. Enquanto HTML estrutura o conteúdo e CSS cuida do estilo, o JavaScript altera conteúdo, responde a eventos e comunica com servidores.

  • Sintaxe básica: variáveis, funções, decisões (if), loops.
  • Trabalha no navegador (DOM) e no servidor (Node.js).
  • Interatividade: manipulação de elementos, formulários e requisições.

Como testar

  1. Salve este arquivo como index.html.
  2. Abra no navegador (duplo clique ou arraste para a janela do navegador).
  3. Abra o console (F12 → Console) para ver mensagens.

Próximos passos

Quando já entender o básico, experimente:

  • Manipular listas de elementos (querySelectorAll, forEach).
  • Fazer requisições com fetch() para obter dados de uma API.
  • Aprender sobre Promises e async/await.
